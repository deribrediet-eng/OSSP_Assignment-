My project explores how the Android operating system can be installed and run on a personal computer using virtualization. So Instead of relying on a physical phone, the idea is to create a virtual environment where Android behaves like a real system.
At first, this might sound complex, I know I have been there 😊. But the concept is simple - one computer, multiple operating systems, all running side by side. This project was more than just an installation task. It provided a deeper understanding of how operating systems work, especially in virtual environments. This makes it possible to experiment, test, and learn without risking the main system. It is great knowing new techniques , so without a further ado go ahead and explore the mobile’s  Android through the Windows of a PC 😊
